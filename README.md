# STEPS

## ENVIRONMENT VARIABLES

* Create `.env` file in the root folder.
* Add the following values: 

``` 
REACT_APP_API_KEY=[YOUR_omdbapi_KEY]
REACT_APP_SERVER_URL=http://www.omdbapi.com

```

## RUN APP

```
npm start
```
